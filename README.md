Project Overview

The Student Marks Management System is a web-based application designed to manage student academic records efficiently.

It allows users (admin/teachers) to add, update, delete, and analyze student marks with a clean and user-friendly interface.

🚀 Features

✨ Add new student details
✨ Update marks and records
✨ Delete student data
✨ View student performance
✨ Search by name or ID
✨ Simple and responsive UI

🛠️ Tech Stack
Category	Technology
Frontend	HTML, CSS, JavaScript
Backend	Python (Flask)
Database	SQLite / MySQL
Tools	VS Code, Git, GitHub
